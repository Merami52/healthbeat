# HealthBeat

**healthbeat** — учебный проект, демонстрирующий основы отказоустойчивости, мониторинга и устранения сбоев.

## Цели проекта
- Разработать простой сервис на **FastAPI**
- Настроить мониторинг через **Prometheus**
- Реализовать health-check и симуляции отказов
- Отработать навыки детектирования и устранения проблем

## План разработки
1. Базовый сервис и контейнеризация
2. Метрики и алертинг
3. Отказоустойчивость и тестирование
